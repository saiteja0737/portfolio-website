# 🌐 Portfolio Website

This repository contains my personal portfolio website, built to organize and present my work in data analytics, machine learning, NLP, and AI-based applications.

🔗 Live Site: https://saiteja0737.github.io/portfolio

---

## 👨‍💻 About

I created this portfolio to bring together my projects, including academic work and personal projects, in one place. It highlights my experience in data analytics, machine learning, and applied AI.

The goal was to keep the website simple, clean, and focused on clearly presenting my work.

---

## 🚀 Features

- Structured presentation of projects
- Sections covering analytics, machine learning, and AI applications
- Direct links to GitHub repositories
- Deployed using GitHub Pages

---

## 🛠️ Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Deployment:** GitHub Pages (via GitHub Actions)
